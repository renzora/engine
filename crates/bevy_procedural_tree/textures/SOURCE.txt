Vendored from the upstream bevy_procedural_tree repository
(assets/textures/deciduous_leaves/) and embedded into the crate binary via
include_bytes! so the default leaf material works without shipping loose asset
files. color = base color (RGBA, alpha = leaf cutout), normal = OpenGL normal
map, roughness = roughness map.
